# SQLite Users Database (Python)

## Завдання
Створити базу даних SQLite за допомогою Python та таблицю для зберігання
облікових записів користувачів.

## Структура таблиці users
- login – імʼя користувача
- password – зашифрований пароль (SHA-256)
- full_name – повне ПІБ користувача

## Використані технології
- Python 3
- sqlite3
- hashlib
